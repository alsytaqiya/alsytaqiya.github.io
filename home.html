Github User App:
Bangkit Submission

GithubUserApp, developed in Kotlin, demonstrates fundamental Android concepts by showcasing a list of Github users along with their respective information. Key features include displaying a minimum of 10 items on the application page using RecyclerView, presenting avatar and user details on the User Detail page, utilizing Parcelable for inter-Activity data transfer, and arranging RecyclerView list items using ConstraintLayout.

Find Me On Github:
Bangkit Submission

FindMeOnGithub is a Kotlin app designed for beginners in Android development. It enables users to search for Github profiles and access essential information such as username, followers, and following counts. The app utilizes RecyclerView for displaying search results and Tab Layout with ViewPager for easy navigation between follower and following lists. Additionally, it includes a loading indicator during data retrieval.

Story App:
Bangkit Submission

Story App is an app built in Kotlin that implement intermediate level Android theories. User can make stories using camera or gallery with additional such as user login, theme and language setting. The app features authentication page with login and registration functionalities, story listing, story addition with image upload, animations, map display, paging, and unit tests for ViewModel functions handling Paging data lists.
